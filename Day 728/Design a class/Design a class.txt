Design a class as described below.

class name : MyClass

function: function name - display(), 
parameters - none, return type - void, 
access modifier - public, function body 
- should print "Hello World"

Note: The driver's code will call the 
display function of the MyClass


Example:

Input: None

Output: "Hello World"


Time Complexity: O(1)
Auxiliary Space: O(1)
