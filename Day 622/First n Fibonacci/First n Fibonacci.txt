Given a number n, return an array containing 
the first n Fibonacci numbers.

Note: The first two numbers 
of the series are 0 and 1.


Examples:

Input: n = 5

Output: [0, 1, 1, 2, 3]


Input: n = 7

Output: [0, 1, 1, 2, 3, 5, 8]


Input: n = 2

Output: [0, 1]


Constraints:
1 <= n <= 30
