Give a n * n square matrix mat[][], 
return all the elements of its 
anti-diagonals from top to bottom.


Examples :

Input: n = 2, mat[][] = [[1, 2],
                        [3, 4]]
Output: [1, 2, 3, 4]

Explanation: 


Input: n = 3, mat[][] = [[1, 2, 3],
                       [4, 5, 6],
                       [7, 8, 9]]

Output: [1, 2, 4, 3, 5, 7, 6, 8, 9]

Explanation: 


Constraints:

1 ≤ n ≤ 10^3
0 ≤ mat[i][j] ≤ 10^6
