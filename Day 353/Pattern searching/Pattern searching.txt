Given a string str and a pattern pat. 

You need to check whether the pattern is 
present or not in the given string. 


Example 1:-

Input:

2
abcdefh
bcd
axzy
xy

Output:
Present
Not present
 

Your task:

You need to complete the function searchPattern() 
and you need to return a boolean value (True or False) 
according to the pattern present or not in the given string.

Returning True will represent the "Present" 
and False will be "Not Present".
 

Constraints:

1 <= T <= 100
1 <= |str| <= 100
1 <= |pat| <= 100
